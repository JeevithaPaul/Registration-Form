# 📋 Registration Form with Real-Time Validation

## 📖 Overview
This project is a simple **Registration Form** built using **HTML, CSS, and JavaScript**. It performs **real-time form validation** to ensure users enter valid information before submitting the form. The submit button remains disabled until all input fields meet the validation requirements.

---

## ✨ Features

- 👤 Name field validation (cannot be empty)
- 📧 Email format validation using Regular Expressions (Regex)
- 🔒 Password validation (minimum 6 characters)
- ⚡ Real-time validation while typing
- 🚫 Submit button disabled until all fields are valid
- 💬 Instant error messages for invalid input
